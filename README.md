# Kid Dice

A small, installable dice roller for family board games.

## Features

- Quick rolls for 1, 2, 3, or 4 standard d6 dice
- Risk-style attacker vs defender rolls with sorted comparison and losses
- Custom dice pools up to 12 dice and 100 sides
- Local roll history
- Offline support through a service worker
- Home screen install support on iOS and Android

## Local Use

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

This repo includes a GitHub Pages workflow. Push to `main`, then enable GitHub Pages with source `GitHub Actions` in the repository settings if it is not enabled automatically.
